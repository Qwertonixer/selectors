Типы селекторов:

  • Cелекторы по типу:

    • Выбирает все элементы с указанным тегом (element { style properties });

  • Универсальній селектор:

    • Звёздочка (*) - универсальный селектор для CSS. Соответствует любому тегу;

  • Селектор классов:

    • Селекторы по CSS классу находят элементы с нужным классом (.classname { style properties });

  • Селектор идентификаторов (id):

    • CSS-селекторы по ID производят выборку всех элементов по ID, полностью совпадающих с селектором (#id_value { style properties });

  • Селектор атрибутов:

    • Селекторы атрибутов отбирают элементы по наличию атрибута или его значению ( [attr] ).
